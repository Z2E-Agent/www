## Tech Stack

- **Framework**: Next.js 16
- **UI Library**: React 19
- **Documentation**: Fumadocs (MDX-based documentation template)
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript
- **Deployment**: Vercel

## Getting Started

```bash
# Install dependencies
bun install

# Run development server
bun run dev

# Build for production
bun run build
```

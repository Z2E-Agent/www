## Feature Modules

Z2E is composed of multiple specialized modules, each designed to handle specific aspects of the penetration testing workflow.

---

| Module | Description |
|--------|-------------|
| `cvemgr` | Custom CVE API handling discovery of existing vulnerabilities |
| `exploitdb-integration` | Ingest data from ExploitDB for real-time exploit search |
| `nikto`, `nmap`, `sqlmap`, etc | Built-in wrappers around standard offensive tools |
| `mcp-protocol-ai-core` | Machine learning protocol core for pattern-based vulnerability guessing |
| `xsshunt-ai-module` | Automates XSS discovery using heuristic mutation rules |
| `crtsh-dnssearcher` | Subdomain discovery using certificate transparency logs |
| `wayback-recon-engine` | Fetches historic page versions via Wayback Machine API |

## Core Repositories

### z2e (Main CLI)

The primary Go-based CLI agent.

```
z2e/
├── cmd/
│   └── z2e/          # Main entry point
├── internal/
│   ├── recon/        # Reconnaissance modules
│   ├── exploit/      # Exploitation engine
│   └── report/       # Report generation
├── pkg/
│   └── mcp/          # AI protocol core
└── configs/          # Default configurations
```

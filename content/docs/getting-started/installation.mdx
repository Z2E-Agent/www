## Prerequisites

Before installing Z2E-Agent, ensure you have the following:

- **Go** - Required for the CLI agent
- **Docker** - For containerized deployment
- **Git** - For cloning repositories

<Callout type="info">The Z2E-Agent isn't finished yet, so this is just a preview of the feature.</Callout>

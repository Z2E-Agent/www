## Prerequisites

Before installing Z2E, ensure you have the following:

- **Go** - Required for the CLI agent
- **Docker** - For containerized deployment
- **Git** - For cloning repositories

<Callout type="info">Z2E-agent aren't finished yet. so just preview of the feature.</Callout>

## Quick Install

```bash
# Clone the repository
git clone https://github.com/Z2E-Agent/z2e-agent.git
cd z2e-agent

make run
```

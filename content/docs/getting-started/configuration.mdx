## Configuration

Z2E uses a YAML-based configuration system for flexibility and reproducibility.

---

## Config File Location

```bash
# Default location
~/.z2e/config.yaml

# Project-specific
./z2e.config.yaml
```

---

## Basic Configuration

```yaml
# z2e.config.yaml
project:
  name: "my-pentest"
  output_dir: "./results"

target:
  scope:
    - "*.example.com"
  exclude:
    - "admin.example.com"

ai:
  enabled: true
  model: "mcp-core-v2"
  confidence_threshold: 0.85

modules:
  recon:
    passive: true
    active: false
  exploitation:
    sandbox_only: true
```

---

## Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `Z2E_API_KEY` | API key for threat intel feeds | - |
| `Z2E_LOG_LEVEL` | Logging verbosity | `info` |
| `Z2E_OUTPUT_DIR` | Default output directory | `./output` |
| `Z2E_SANDBOX_MODE` | Force sandbox mode | `true` |
